# Vite 代理配置项目

这是一个带有代理配置的 Vite.js 项目。

## 功能特点
- 基于 Vite.js 构建
- 支持代理配置
- 支持 Docker 部署

## 快速开始

### 环境要求
- Node.js
- pnpm

### 安装步骤
1. 克隆仓库
2. 安装依赖：
```bash
pnpm install
```

### 开发
运行开发服务器：
```bash
pnpm dev
```

### 构建
构建生产版本：
```bash
pnpm build
```

### Docker 部署
使用 Docker 构建和运行：
```bash
docker-compose up
```
