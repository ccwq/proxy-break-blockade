# Vite Proxy ABC

This is a Vite.js project with proxy configuration.

## Features
- Built with Vite.js
- Proxy configuration
- Docker support

## Getting Started

### Prerequisites
- Node.js
- pnpm

### Installation
1. Clone the repository
2. Install dependencies:
```bash
pnpm install
```

### Development
Run the development server:
```bash
pnpm dev
```

### Building
Build for production:
```bash
pnpm build
```

### Docker Support
Build and run with Docker:
```bash
docker-compose up
```
